-------------------------------------------------------------------------
                  『悪女の栄冠』マニュアル
-------------------------------------------------------------------------


■はじめに

この度は『悪女の栄冠』をプレイして頂き、誠にありがとうございます。
このゲームは、ツール『LiveMaker』を使って開発された１８禁ADVゲームです。

注意：お使いのハードディスクにmsg0008フォルダをコピーして
　　　akujo.exeを起動してください。
　　　CDからの直接起動は不可となっております。


■動作環境

OS     : Windows7/8/10
CPU    : Pentium300MHz以上
メモリ : 128MB以上
画面   : 解像度は1280x720以上 色数はハイカラーまたはフルカラー

スペックが低い環境の場合、ムービー等の
一部動作が遅くなる場合があります。

■ゲームシステムの説明

【文章表示中の操作】

　右クリック・ESCキー：
　　システム画面を表示します
　左クリック・エンターキー：
　　文章を進めます
　CTRLキー：
　　押している間、文章を高速表示します
　スペースキー：
　　文章部分の表示/非表示を切り替えます
　ホイール・↑↓キー：
　　メッセージボックス内にテキスト履歴を表示します。
　　ボイスアイコンがついているテキストは
　　リピート再生可能です。

【ボイスリピートについて】
　本編進行中はメッセージボックスメニューの
　「Voice」ボタンにて進行中のボイスを
　リピート再生できます。
　回想モードでは使用不可となっておりますので、
　ホイールもしくは↑↓キーのテキスト履歴にて
　リピート再生してください。


【システム画面】

　○文字を消す：
　　文章部分を画面から消去します
　○シナリオ回想：
　　表示済みの文章を見ることができます
　○読んだ文章を飛ばす：
　　一度読んだ文章を高速表示します
　○自動テキスト送り：
　　クリックしなくても、自動的に文章が進みます
　○セーブ：
　　現在のゲームの状態を保存します
　○ロード：
　　セーブしたデータを読み込みます
　○オプション：
　　○読んだ文章を自動的に飛ばす：
　　　チェックを付けると、一度読んだ文章が表示された時、自動的に高速表示に切り
　　　替わります
　　○テキスト速度：
　　　文章が表示される速度を調節します
　　○[自動テキスト送り]時間設定：
　　　「自動テキスト送り」で次のページに進むまでの待ち時間を指定します
　　○フォント選択：
　　　文章用フォントを選択します
　　○サウンドを再生する：
　　　チェックを付けると、サウンドが鳴ります
　　○音量調節：
　　　音量を調節します
　　○フルスクリーン：
　　　ウィンドウ、フルスクリーンを切り替えます
　　○ディスプレイモード：
　　　フルスクリーン時のディスプレイモードを選択します
　○ゲーム終了：
　　○タイトル画面に戻る：
　　　起動時に表示されるタイトル画面に戻ります
　　○終了：
　　　ゲームを終了します：


【起動時の操作】

前回終了時にフルスクリーンだった場合は、次に起動した時、自動的にフルスク
リーンになります。
これを回避したい場合は、CTRLキーを押した状態で起動してください。
